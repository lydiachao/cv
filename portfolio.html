My Top 10 Movies Project
    I completed this project using several tools in Python:
  • Flask, Flask-WTF, wtforms, and Flask-Bootstrap to create beautiful Bootstrapped HTML templates for the pages;
  • Flask-SQLAlchemy to create the movie database to add, save, update, and delete movies from the collection of movies as well as to rank movies based on Ratings given to each movie;
  • API requests from the Movie Database to search and get movie titles, release dates, descriptions/overviews, and poster pictures to display for each movie selected and added to the database.

    • Note: For the purpose of this portfolio, the buttons have been inactivated and HTML code hardcoded. In the original project, the templates were autogenerated from the python code. The update buttons sent back information to the Flask server about which movie was selected for update, and redirect the user to the correct HTML page to add or update their movie reviews and ratings for the movie selected. The Add button lets the user call the Movie Database API to search and get the list of movies with the same titles, from which the user would select the right movie to add to the database/collection. The Delete button is used to delete a movie from the database if the user changes their mind.
