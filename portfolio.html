My Top 10 Movies Project
    I completed this project using several tools in Python:

  • Flask, Flask-WTF, wtforms, and Flask-Bootstrap to create beautiful Bootstrapped HTML templates for the pages;
  • Flask-SQLAlchemy to create the movie database to add, save, update, and delete movies from the collection of movies as well as to rank movies based on Ratings given to each movie;
  • API requests from the Movie Database to search and get movie titles, release dates, descriptions/overviews, and poster pictures to display for each movie selected and added to the database.

    • Note: For the purpose of this portfolio, the buttons have been inactivated and HTML code hardcoded.
